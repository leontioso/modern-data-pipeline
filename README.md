## 🚀 Quick Start

```bash
# Setup environment
cd ~/data-engineer-portfolio
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Start infrastructure
docker-compose up -d

# Run pipeline
python src/main_pipeline.py
```
